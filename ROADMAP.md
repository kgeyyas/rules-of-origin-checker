# Roadmap

Built with **GAP OS** using **The Signal Hunter Framework**.

Proveni is an open-source Rules of Origin decision engine that helps businesses determine whether products qualify for preferential tariff treatment under Free Trade Agreements (FTAs).

Rather than simply checking eligibility, Proveni is designed to explain *why* a product qualifies—or does not qualify—and provide actionable guidance for improving origin compliance.

---

# Phase 1 — Core Foundation (Current)

## Goal

Build a transparent and easy-to-use Rules of Origin checker.

### Features

- Public open-source release
- Country of export selection
- Country of import selection
- FTA selection
- HS Code lookup
- Product eligibility check
- Product Specific Rules (PSR) lookup
- Origin qualification result
- Explainable decision output
- Responsive UI
- MIT License
- Documentation

---

# Phase 2 — Product Maturity

## Goal

Support real-world origin qualification workflows.

### Features

- Tariff Shift analysis (CC / CTH / CTSH)
- Regional Value Content (RVC) calculations
- Wholly Obtained determination
- De Minimis rule support
- Cumulation rule support
- Bill of Materials (BOM) evaluation
- Multiple qualification methods
- Origin explanation report
- PDF export
- CSV import & export

---

# Phase 3 — Intelligence & Automation

## Goal

Turn origin qualification into an intelligent decision-support system.

### Features

- Multi-FTA comparison
- AI-assisted Rules of Origin interpretation
- Origin optimization recommendations
- Alternative sourcing suggestions
- Supplier origin validation
- REST API
- Community contributions
- Performance optimization

---

# Phase 4 — GAP OS Integration

## Goal

Connect origin intelligence with the broader GAP OS ecosystem.

### Features

- Automatic HS Code retrieval from HA.RS.QO
- Tariff simulation with Tarifio
- Landed Cost integration with Netto
- Supplier compliance integration with Compila
- Cross-tool workflows
- Shared trade intelligence
- AI feedback learning
- Unified trade decision engine

---

# Long-Term Vision

Proveni is evolving beyond a Rules of Origin checker.

The long-term vision is to become an intelligent origin reasoning engine capable of explaining qualification logic, identifying compliance risks, recommending sourcing improvements, and supporting audit-ready trade decisions across global supply chains.
